# Brainstem - Chrome Extension

Save links and highlighted text with a simple right-click.

## Features

- Right-click any link to save
- Highlight text and right-click to save with URL
- All data stored locally in your browser

## Installation

1. Download this extension
2. Go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the extension folder

## Usage

**Save a link:** Right-click any link → "Save to Note"

**Save text:** Highlight text → Right-click → "Save to Note"

**View notes:** Click the extension icon in toolbar

## Tech Stack

- Manifest V3
- HTML & CSS & Vanilla JavaScript
- Chrome Storage API
- Chrome Context Menus API
